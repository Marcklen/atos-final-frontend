Projeto final ATOS Aplicação Fullstack ATOS

Olá! Seja bem vindo ao sistema de controle e separação de pedidos desenvolvido por mim no curso de Java da Atos em parceria com a UFN.
Esse é um sistema para gerenciamento de frente de loja simulando o processo de organização e informações desenvolvido em Angular 12 e Spring Boot.
O aplicação front está na nuvem com Vercel e o back foi feito a tentativa na Heroku, porém foi solicitado um cartão de crédito e achei melhor subir somente o codigo para o GitHub.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
