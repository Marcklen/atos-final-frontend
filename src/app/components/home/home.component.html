Projeto Final Atos Separação de Pedidos - Almoxarifado

Olá! Seja bem vindo ao sistema automatizado de separação de pedidos desenvolvido por mim no curso de Java da Atos em parceria com a UFN.
Esse é um sistema para gerenciamento de almoxarifado simulando um simples processo de logistica desenvolvido em Angular 12 e Spring Boot.
O sistema está rodando na nuvem sendo o front na Vercel e o back foi feito a tentativa na Heroku, porém foi solicitado um cartão de crédito e achei melhor subir somente o codigo para o GitHub.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
